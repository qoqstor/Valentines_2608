## 怎麼用？

### 📝 寫一封文字告白

1. 點畫面中央的信封 💌
2. 選「文字告白」
3. （選填）填上想告白的對象，例如「三班的她」
4. 寫下你的告白內容
5. 按「投稿」

送出後，你的告白會以動物代號出現在信牆上，完全看不出是誰。

### 🎤 錄一段語音告白

1. 點信封 💌 → 選「語音告白」
2. （選填）填上告白對象
3. 按「開始錄音」，說出你想說的話（最長 **20 秒**）
4. 說完按「停止」，稍等幾秒，系統會自動把你的話轉成文字
5. 確認沒問題後按「確認，送出投稿」

> 💡 錄音時盡量在安靜的地方、對著手機清楚地說，辨識會更準。

### 💻 筆電用戶必看：錄音前先關閉「麥克風音效」

部分筆電（尤其 ASUS / 使用 Realtek 音效的機種）內建的麥克風「智慧音效」會干擾錄音，造成**開頭大聲、後面悶掉或只剩雜音**。錄音前請先做一次以下設定（只需設定一次）：

![麥克風設定示意圖](mic-setting.svg)

1. 開啟 **Realtek Audio Console / MyASUS 音效主控台**（開始選單搜尋「Realtek」或「音效主控台」）
2. 進入 **麥克風** 頁面，找到 **「關閉麥克風音效」** 的開關，切成 **開啟（原始音訊）**
3. **完全關閉瀏覽器再重開**，回到網站錄一段測試

找不到上面的面板？改走 Windows 內建路徑：
工作列喇叭右鍵 → **音效設定** → **更多音效設定** → **錄製** → 麥克風 → **內容** → **增強功能** → 勾選 **停用所有音效** → 確定。

> 錄音時如果畫面出現「⚠️收音品質過差」的提示，就代表這個設定還沒生效，請再檢查一次。

### 💭 猜猜這是誰寫的

💡 點「目前已有 X 封投稿信」可以打開完整投稿清單，逐封瀏覽。

語音告白播放時會**隨機變聲**（音調會變高或變低），沒有人聽得出是你的聲音 🎭

### 💭 猜猜這是誰寫的

點開任何一封告白，下面有「猜猜這是誰寫的？」，輸入你猜的名字送出。

## 常見問題

**Q：別人會知道是我寫的嗎？**
完全不會。你只會顯示成「🐾 匿名○○」這樣的動物代號，沒有任何真實身分資訊。

**Q：我可以投稿幾封？**
想投幾封都可以，盡情表白 💕

**Q：筆電錄音破音、悶悶的、或只有雜音？**
請看上面「💻 筆電用戶必看」，把麥克風音效關閉（原始音訊）再重開瀏覽器，多數情況就會恢復正常。

**Q：語音辨識出來的字怪怪的怎麼辦？**
送出前可以先看轉出來的文字，如果不太對，可以按取消重錄一次。安靜環境、講清楚一點會更準。

**Q：手機和電腦都能用嗎？**
都可以！用手機或電腦的瀏覽器打開網址即可，不用下載任何 App。

**Q：需要註冊或登入嗎？**
不用。打開就能直接使用。

**Q：右上角的「音樂」是什麼？**
點一下會播放背景音樂，讓氣氛更浪漫 🎵 不想聽再點一下就好。
